The ExcelStandardModelProvider extends the StandardModelProvider cannot be used alone. The ExcelStandardModelProvider reads class data from an excel file and adds them to the model of the StandardModelProvider. The Excel data may represent one or more complete new classes or new attributes to classes existing in the standard model.

To use this functionality add the MetaInfo “metainfo useExtensionPlugin ExcelStandardModelProvider” to your standard model. This will cause the StandardModelProvider to trigger the ExcelStandardModelProvider reading the excel file defined in the default input file _excel.properties of the ExcelStandardModelProvider. The comment of this file provides basic information about read data from Excel files. For details see https://github.com/iks-github/MOGLiCodeCreator/wiki/Using-the-ExcelStandardModelProvider.
